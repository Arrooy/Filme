Filme es un Chatbot dissenyat per a
resoldre dubtes i oferir informacio
sobre el mon cinematografic.
S'ha desenvolupat en Java fent servir
l'IDE IntelliJ IDEA. Es recomana el
seu ús per a la correcta execució.

Per tal de fer servir el Chatbot només
caldrà executar l'arxiu Brain.java,
que automàticament obrirà una finestra
grpafica on xatejar. A partir d'aquí
ja es pot fer servir el chatbot amb
normalitat.
